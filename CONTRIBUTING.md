# Contributing

## Workflow
- Usamos GitHub Flow: https://docs.github.com/en/get-started/using-github/github-flow
- Rama: `feat/<topic>` o `fix/<topic>`
- (Opcional) Commits: Conventional Commits

## Reglas
- No merge si CI falla.
- No se implementa si el doc/ADR está TBD.
- Cambios a API requieren actualizar `/docs/02-api/openapi.yaml`.

## PR
Usa la plantilla en `.github/PULL_REQUEST_TEMPLATE.md`.
