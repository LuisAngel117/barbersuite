# Visión

BarberSuite es un SaaS multi-tenant para barberías:
- Un tenant = barbería
- Múltiples sucursales
- Operación diaria: agenda, empleados, clientes, caja, reportes y notificaciones

Objetivo de portafolio:
- arquitectura SaaS real, seguridad, calidad y delivery (PR + CI)
- full-stack (API + Next.js)
